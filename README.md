# UI Localization - Fuel Cost Calculator

## Description
JavaFX application that calculates fuel consumption and trip cost with multi-language support.

## Features
- Calculate fuel needed and total cost
- Supports 4 languages:
    - English
    - French
    - Japanese
    - Persian

## Technologies
- JavaFX
- Maven
- Jenkins (CI/CD)
- Docker

## How to Run
```bash
mvn javafx:run
